# **OpenCV介绍**

OpenCV 是一个开源的计算机视觉库。
OpenCV 库由 C 和 C++语言编写，涵盖计算机视觉各个领域内的 500 多个函数，可以在多种操作系统上运行。它旨在提供一个简洁而又高效的接口，从而帮助开发人员快速地构建视觉应用。
